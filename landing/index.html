Requisitos do Sistema

  • Windows 10/11 (64-bit)
  • 4GB de RAM (8GB recomendado)
  • 500MB de espaço em disco
  • Conexão com internet

Aviso do Windows SmartScreen

Como o Openwork ainda nao possui assinatura digital, o Windows pode exibir um aviso de seguranca. Siga estes passos para prosseguir com a instalacao:

1

Aviso aparecera

"O Windows protegeu seu computador" sera exibido na tela.

2

Clique em "Mais informacoes"

O link aparece abaixo da mensagem de aviso. Clique nele para ver mais opcoes.

3

Clique em "Executar assim mesmo"

O botao aparecera apos clicar em "Mais informacoes". A instalacao prosseguira normalmente.
